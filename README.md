# 96BoardsGPIO_setup
Install script for 96BoardsGPIO

### Update the installed image
```
$ sudo apt-get update
$ sudo apt-get upgrade
$ sudo apt-get dist-upgrade
```

### Run setup script
```
$ chmod +x install-GPIO-db410c.sh
$ ./install-GPIO-db410c.sh
```
